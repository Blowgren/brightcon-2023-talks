# Environments

List environments:

    conda info --envs

Remove an environment:

    sudo conda remove --name <name> --all

    sudo conda create -n premise -c conda-forge -c cmutel -c romainsacchi premise"==2023.09.11" brightway2"==2.4.3" ipykernel
